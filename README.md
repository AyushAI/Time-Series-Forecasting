# 📈 Time Series Forecasting Projects

A collection of Time Series Forecasting projects, statistical analyses, handwritten study notes, and Python implementations covering real-world datasets and forecasting techniques.

## Repository Contents

### ✈️ Airline Passenger Time Series

* Time series analysis and forecasting of airline passenger data.
* Implemented using ARIMA models.
* Includes trend, seasonality, and forecasting analysis.

### 💳 Credit Card Transaction Forecasting

* Forecasting transaction patterns using SARIMAX.
* Incorporates external variables to improve prediction accuracy.
* End-to-end statistical modeling workflow.

### 🚗 Tesla Shares Forecasting

* Stock price trend analysis using Moving Average and statistical techniques.
* Exploratory data analysis and forecasting insights.
* Focus on understanding financial time series behavior.

### 🚦 Traffic Time Series Analysis

* Traffic volume forecasting using SARIMA models.
* Seasonal pattern detection and future traffic predictions.
* Useful for transportation and demand forecasting studies.

### 📝 Handwritten Notes

* Personal handwritten notes covering Time Series Forecasting concepts.
* Statistical foundations, formulas, model assumptions, and practical insights.
* Helpful for interview preparation and revision.
